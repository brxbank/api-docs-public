# Onboarding Guide

Follow this step-by-step guide to create your account, submit documents, and start operating with the BRZ PIX API.

## Step 1 — Create Account

Create your account with `POST /v1/auth/signup`. Save the returned `keyId` and `secretKey` — you will need them later.

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "phone": "+5511999998888",
    "password": "your-secure-password",
    "name": "Your Name",
    "document": "12345678901"
  }'
```

OTP codes will be sent via email and SMS.

## Step 1b — Verify Email

Verify your email with the OTP code received:

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/verify-email \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "code": "123456"
  }'
```

## Step 1c — Verify Phone

Verify your phone with the OTP code received via SMS:

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/verify-phone \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "+5511999998888",
    "code": "654321"
  }'
```

## Step 1d — Resend OTP (Optional)

If needed, resend the OTP code (maximum 5 per hour):

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/resend-otp \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "type": "email"
  }'
```

## Step 2 — Login

Log in to obtain a session token:

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "password": "your-secure-password"
  }'
```

## Step 3 — Submit Documents

Document submission involves three stages:

### 3a. Get Pre-signed URLs

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/documents/presigned-urls \
  -H "Authorization: Bearer <session-token>" \
  -H "Content-Type: application/json" \
  -d '{
    "documents": ["cpf_frente", "cpf_verso", "selfie_com_cpf", "comprovante_residencia"]
  }'
```

### 3b. Upload Files

Upload each file directly to the S3 URL (raw binary, no FormData):

```bash
curl -X PUT "<presigned-url>" \
  -H "Content-Type: image/jpeg" \
  --data-binary @document.jpg
```

### 3c. Confirm Uploads

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/documents/confirm \
  -H "Authorization: Bearer <session-token>" \
  -H "Content-Type: application/json"
```

### Required Documents

**Individual (Pessoa Física):**
- `cpf_frente` — CPF front
- `cpf_verso` — CPF back
- `selfie_com_cpf` — Selfie holding CPF
- `comprovante_residencia` — Proof of address

**Company (Pessoa Jurídica):**
- `cnpj` — CNPJ document
- `contrato_social` — Articles of incorporation
- `comprovante_residencia_empresa` — Company proof of address

## Step 4 — Wait for Approval

Check your account status:

```bash
curl -X GET https://pix-api.corpx.com.br/v1/auth/status \
  -H "Authorization: Bearer <session-token>"
```

Wait for the team to approve your account.

## Step 5 — Generate Bearer Token

After approval, generate your Bearer Token:

```bash
curl -X POST https://pix-api.corpx.com.br/v1/auth/bearer-token \
  -H "Content-Type: application/json" \
  -d '{
    "key_id": "your-key-id",
    "secret_key": "your-secret-key"
  }'
```

## Step 6 — Register a PIX Key

Register a PIX key at DICT/BACEN:

```bash
curl -X POST https://pix-api.corpx.com.br/api/v1/pix/keys \
  -H "Authorization: Bearer <your-jwt>" \
  -H "Content-Type: application/json" \
  -d '{
    "key_type": "EVP"
  }'
```

<Callout intent="tip">
  For EVP (random) keys, just send `{"key_type": "EVP"}`. BACEN generates the key automatically, and account data is auto-filled from your onboarding.
</Callout>

## Step 7 — Ready to Operate!

You can now create charges (Cash-In), make transfers, and check your balance. Explore the API Reference for all available endpoints.
